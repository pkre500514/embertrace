## Description

EmberTrace is an evidence-grounded incident-report drafting demo for fire crews.

The demo cross-checks simulated helmet-camera, radio, scene-photo, CAD, and crew-clarification records to create a reviewable draft. Every supported claim links back to its source. Unknown information stays visible for human review, and the prototype never auto-submits, determines origin or cause, or uses real emergency-service data.

Built with Codex and GPT-5.6. The project runs locally with no API key, account, upload, or external model call.
