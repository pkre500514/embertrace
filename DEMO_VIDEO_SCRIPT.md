## 제출 설명 한 문단

> EmberTrace helps fire crews transform post-incident evidence into a grounded, reviewable report draft. It connects each supported statement to captured records, preserves unresolved fields for human confirmation, and never auto-submits or determines origin and cause. The demo runs locally with simulated, de-identified evidence and no API key.
